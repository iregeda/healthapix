INTRODUCTION
------------

This module provide Field Element support to upload image or browser
existing image from server like "media" field type in Drupal 7.

 * For a full description of the module visit:
   https://www.drupal.org/project/image_browser

 * To submit bug reports and feature suggestions, or to track changes visit:
   https://www.drupal.org/project/issues/image_browser


REQUIREMENTS
------------

This module requires no modules outside of Drupal core.

The Image Entity Browser module uses Entity Browser.

 * Entity Browser - https://www.drupal.org/project/entity_browser


INSTALLATION
------------

 * Install the Image Entity Browser module as you would normally install a
   contributed Drupal module. Visit https://www.drupal.org/node/1897420 for
   further information.


CONFIGURATION
-------------

  1. Navigate to Administration > Extend and enable the Image Entity Browser
     module and its dependencies.
